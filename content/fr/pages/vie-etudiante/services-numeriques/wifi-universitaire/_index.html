Les bornes WiFi de l'IUT sont en cours de remplacement. Le WiFi sera indisponible le jeudi 12 septembre pendant la phase de migration vers les nouvelles bornes. Des perturbations pourront persister le vendredi 13 septembre.

notes: >- alt: >- credit: >- - kind: heading title: >- Le WiFi à l'IUT slug: "le-wifi-a-l-iut" position: 0 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Le réseaux WiFi "eduroam" est accessible, grâce à vos identifiants, dans toutes les universités participants au projet.

Les avantages du réseaux "eduroam" :

  • connexion sécurisée
  • connexion automatique
  • accès au WiFi "eduroam" dans les universités en France et à l'étranger
notes: >-

Plus d'informations : www.eduroam.fr

Pour vous identifier sur le réseau eduroam, vous devez ajouter le domaine de votre établissement à votre identifiant.
Par exemple : pour le compte informatique jdupond002, l'identifiant eduroam sera jdupond002@u‑bordeaux‑montaigne.fr

    • Apple App Store
  2. Sélectionnez votre établissement Université Bordeaux Montaigne en tappant le nom dans le champ de recherche
  3. Cliquez sur "Connecter"
  4. Entrez vos identifiants de l'université (avec le domaine @u-bordeaux-montaigne)
