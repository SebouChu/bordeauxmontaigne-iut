L'inscription administrative permet de confirmer ou de modifier les informations dont l'établissement a besoin pour assurer le suivi de votre cursus universitaire. C'est donc une étape obligatoire, même si vous étiez déjà inscrit·e l'année précédente.

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

A noter :
Vous êtes définitivement inscrit·e à l'Université Bordeaux Montaigne une fois que vos certificats de scolarité et votre carte étudiante vous ont été délivrés. Ils vous seront nécessaires pour accéder aux services numériques (outils pédagogiques, bureau virtuel, impression, etc.).

notes: >-

L'inscription et la gestion des étudiants font l'objet d'un traitement de données personnelles

Les personnes qui s'inscrivent en contrat de professionnalisation ou en formation continue ne sont pas concernées par la CVEC.

alt: >- credit: >- - kind: heading title: >- 2. L'inscription administrative slug: "2-l-inscription-administrative" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Une fois que vous êtes admis à rejoindre notre établissement ou à effectuer une nouvelle année, l'inscription administrative est obligatoire afin d'enregistrer vos informations et de régler les frais le cas échéant.

La date et la démarche de l'inscription en ligne diffèrent selon votre cas :

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Ré-inscription ranks: self: 3 data: layout: alt_background text: >-

Si vous êtes déjà inscrit à l'IUT en BUT ou dans une autre formation à l'Université Bordeaux Montaigne, vous devez vous ré-inscrire afin d'actualiser votre dossier.

À partir du 26 juin

notes: >- alt: >- credit: >- - kind: block template: chapter title: >- Nouvelle inscription en BUT1 ranks: self: 3 data: layout: no_background text: >-

Si vous vous inscrivez pour la première fois à l'IUT Bordeaux Montaigne pour intégrer un BUT1 après votre candidature sur Parcoursup, vous devez effectuer votre inscription administrative en ligne à partir de début juillet.

Renseignez le champ "N° de candidature" comme suit : P24 suivi des 7 chiffres de votre n° de dossier ParcourSup (ajoutez un 0 après P24 si votre n° de dossier ne contient que 6 chiffres). Vous devrez également renseigner votre adresse de courriel utilisée sur Parcoursup.
