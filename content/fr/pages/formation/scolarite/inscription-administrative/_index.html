Les inscriptions administratives, dépôts de pièces, ainsi que les réponses aux demandes d'assistance seront interrompues du 20 juillet au 19 août inclus.

notes: >- alt: >- credit: >- - kind: heading title: >- Inscription administrative à partir de fin juin slug: "inscription-administrative-a-partir-de-juillet" position: 1 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

L'inscription administrative permet de confirmer ou de modifier les informations dont l'établissement a besoin pour assurer le suivi de votre cursus universitaire. C'est donc une étape obligatoire, même si vous étiez déjà inscrit·e à l'IUT l'année précédente.

Votre inscription doit être finalisée impérativement avant le 6 septembre.

notes: >- alt: >- credit: >- - kind: block template: timeline title: >- Les étapes pour finaliser votre inscription ranks: self: 3 children: 4 data: layout: vertical events: - title: >- text: >- La Contribution Vie Étudiante et de Campus - title: >- text: >- L'inscription administrative - title: >- text: >- Les pièces justificatives - title: >- text: >- Le paiement des frais - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

A noter :
Vous êtes définitivement inscrit·e à l'Université Bordeaux Montaigne une fois que vos certificats de scolarité et votre carte étudiante vous ont été délivrés. Ils vous seront nécessaires pour accéder aux services numériques (outils pédagogiques, bureau virtuel, impression, etc.).

notes: >-

L'inscription et la gestion des étudiants font l'objet d'un traitement de données personnelles

alt: >- credit: >- - kind: heading title: >- 1. La Contribution de Vie Étudiante et de Campus slug: "1-la-contribution-de-vie-etudiante-et-de-campus" position: 2 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

La CVEC est due chaque année par les étudiant·e·s en formation initiale ou en apprentissage.
L'enregistrement en ligne est obligatoire, que vous soyez exonéré ou non, avant votre inscription administrative.

→ S'acquitter de la CVEC

notes: >-

Les personnes qui s'inscrivent en contrat de professionnalisation ou en formation continue ne sont pas concernées par la CVEC.

alt: >- credit: >- - kind: heading title: >- 2. L'inscription administrative slug: "2-l-inscription-administrative" position: 3 rank: 2 - kind: block template: chapter title: >- ranks: self: false data: layout: no_background text: >-

Une fois que vous êtes admis à rejoindre notre établissement ou à effectuer une nouvelle année, l'inscription administrative est obligatoire afin d'enregistrer vos informations et de régler les frais le cas échéant. Vous devez impérativement finaliser votre inscription avant le 6 septembre.

La date et la démarche de l'inscription en ligne diffèrent selon votre cas ↓

notes: >- alt: >- credit: >- - kind: block template: call_to_action title: >- Réinscription ranks: self: 3 data: layout: accent_background text: >-

Si vous êtes inscrit actuellement à l'IUT en BUT ou avez déjà été inscrit dans une autre formation à l'IUT ou à l'Université Bordeaux Montaigne, vous devez vous réinscrire afin d'actualiser votre dossier.

Munissez-vous de votre numéro d'étudiant pour vous authentifier.

Pour les personnes en attente de leurs résultats de fin d'année à l'IUT, la réinscription sera disponible après le jury, à partir du 8 juillet.

Si vous êtes en prolongation (BUT, DU ou LP), la réinscription est possible à partir du 16 septembre.

alt: >- credit: >- buttons: - title: >- Inscription en ligne url: >- https://www.apogee-montaigne.u-bordeaux.fr/apo3web/etapes/identification.jspx target_blank: true - kind: block template: call_to_action title: >- Nouvelle inscription en BUT1 ranks: self: 3 data: layout: accent_background text: >-

Si vous étiez déjà inscrit·e à l'université Bordeaux Montaigne l'an dernier (par exemple en BUT ou Licence...), vous devez effectuer une réinscription (voir bloc ci-dessus ↑)

Si vous vous inscrivez pour la première fois à l'IUT Bordeaux Montaigne pour intégrer un BUT1 après votre candidature sur Parcoursup, vous devez effectuer votre inscription administrative en ligne à partir de début juillet.

Renseignez le champ "N° de candidature" comme suit : P24 suivi des 7 chiffres de votre n° de dossier Parcoursup (ajoutez un 0 après P24 si votre n° de dossier ne contient que 6 chiffres). Vous devrez également renseigner votre adresse de courriel utilisée sur Parcoursup.

alt: >- credit: >- buttons: - title: >- Inscription en ligne url: >- https://www.apogee-montaigne.u-bordeaux.fr/primo3web/ident1.jsf target_blank: true - kind: block template: call_to_action title: >- Nouvelle inscription autres formations ranks: self: 3 data: layout: accent_background text: >-

Si vous étiez déjà inscrit·e à l'université Bordeaux Montaigne l'an dernier (par exemple en BUT ou Licence...), vous devez effectuer une réinscription (voir bloc ci-dessus ↑)

Si vous intégrez l'IUT via une passerelle BUT ou en licence professionnelle, en DU, vous devez effectuer votre inscription en ligne après avoir été admis sur CandIUT.

Munissez-vous de votre n° de candidature CandIUT (attention, il est différent du n° de candidat  !) ainsi que de l'adresse de courriel que vous avez utilisée lors de votre candidature.

À partir du 15 juillet.
